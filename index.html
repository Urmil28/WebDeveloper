Guardian-NYTimes NewsApp
ReactJs/Bootstrap/Node.js

Web application that allows users to search for news using autocomplete, see news articles, share news articles via facebook, twitter or email. Users can navigate to news articles on click, bookmark news articles and also comment on the articles.

Technology

  • This website Bootstrap3 for a mobile-first and responsive layout, and is built using ReactJS and Bootstrap.
  • On the backend an Express/Node.js app is hosted on Microsoft Azure that acts as a middleware to scrape data from NYTimes and The Guardian using their respective APIs and return the various news articles to the user.
